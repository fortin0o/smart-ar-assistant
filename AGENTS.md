# AGENTS.md

## Available Agents

This project uses specialized agents to improve development quality.

---

# 1. AR Engineer

Role:
Responsible for augmented reality functionality.

Responsibilities:

* Marker detection
* Camera initialization
* AR scene management
* MindAR integration
* AR tracking optimization

Focus:

* Stable tracking
* Mobile compatibility
* Performance

---

# 2. Three.js Engineer

Role:
Responsible for 3D rendering.

Responsibilities:

* Model loading
* Animations
* Materials
* Lighting
* Selection system

Focus:

* FPS optimization
* Clean scene architecture
* Reusable 3D components

---

# 3. AI Engineer

Role:
Responsible for AI interaction.

Responsibilities:

* OpenAI integration
* Context generation
* Prompt engineering
* Response streaming

Focus:

* Context awareness
* Fast responses
* Accurate explanations

---

# 4. Frontend Engineer

Role:
Responsible for UI implementation.

Responsibilities:

* Layout
* Components
* Responsiveness
* State management

Focus:

* Mobile-first design
* Reusable components
* Accessibility

---

# 5. UX Designer

Role:
Responsible for user experience.

Responsibilities:

* Interaction design
* Navigation flow
* Information hierarchy

Focus:

* Simplicity
* Clarity
* Professional feel

---

# 6. Performance Engineer

Role:
Responsible for optimization.

Responsibilities:

* Bundle size reduction
* Lazy loading
* Render optimization
* Asset optimization

Focus:

* Mobile performance
* Fast load times
* Smooth animations

---

# Development Workflow

Before implementing a feature:

1. UX Designer defines user flow.
2. Frontend Engineer designs component structure.
3. AR Engineer handles AR integration.
4. Three.js Engineer handles rendering.
5. AI Engineer integrates assistant features.
6. Performance Engineer reviews implementation.

---

# Priority Order

1. Mobile Experience
2. AR Stability
3. AI Context Accuracy
4. 3D Performance
5. Visual Polish

Never sacrifice performance for visual effects.

The application should feel fast, responsive, and professional on mobile devices.
